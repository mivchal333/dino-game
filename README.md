# Mario Runner Game
![Mario Runner Image](https://github.com/mivchal333/dino-game/blob/master/readme/mario_runner_game.png?raw=true)

## About The Project
Marion Runner game based on Chrome T-Rex runner.

### Build with
* JavaScript
* Phaser
* Webpack

## Instalation
1. Install NPM packages
   ```sh
   npm install
   ```
2. Start dev server
   ```sh
    npm run dev
    ```
3. Enjoy!
